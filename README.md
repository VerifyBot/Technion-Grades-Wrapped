# Technion Grade Wrapped

A web application that generates a personalized, Spotify-style "Wrapped" presentation of your academic semester using your Technion grades data. 

**Privacy Note:** All data extraction and presentation processing happens entirely client-side. Your academic data is never uploaded to or stored on any external servers.

![alt text](download.png)

## How to Use

### 1. Download Your Data (`grades.json`)
To generate your Wrapped presentation, you first need to export your grades from the Technion student portal.

1. Log in to the [Technion Grades Portal](https://grades.technion.ac.il/).
2. Open your browser's Developer Tools (Press `F12`, or `Ctrl+Shift+I` on Windows/Linux, `Cmd+Option+I` on Mac).
3. Navigate to the **Console** tab.
4. Copy the script below, paste it into the console, and press **Enter**:
